# Maps

Overview of the top-level themes for StoryMaps application.

## Maps

| UID | Title | Description |
|-----|-------|-------------|
| maps-001 | Map Management | Core functionality for creating, editing, and deleting story maps |
| maps-002 | Visualization | UI components for viewing and navigating story maps |
| maps-003 | Data API | Backend operations for data import/export and REST API |
| maps-004 | Self-hosted | Deployment and configuration for self-hosted environments |
| maps-005 | Dark Mode | UI enhancement for toggling between light and dark themes |
